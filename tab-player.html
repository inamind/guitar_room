C → B Guitar TAB Player

이미지에 보이는 TAB을 기준으로, 괄호의 3은 첫 3개 세로줄에만 적용되는 셋잇단음표입니다. 즉 X → 음 → X가 한 박자 안에 들어가고, 마지막 음은 그 다음 한 박자 전체를 차지합니다.

C B e|--------------------------------------| B|--x--13--x--13----------x--12--x--12--| G|--x---x--x---x----------x---x--x---x--| D|--x--10--x--10----------x---9--x---9--| A|--------------------------------------| E|--------------------------------------| │ │ │ │ │ │ └─── 3 ───┘ └─── 3 ───┘
현재 코드: 대기
패턴: 첫 3개 = 셋잇단 1박, 마지막 = 1박
기준 튜닝: 1번줄 e4, 2번줄 B3, 3번줄 G3, 4번줄 D3, 5번줄 A2, 6번줄 E2. 화면에는 기타줄 6개를 모두 표시했고, 실제 소리는 첨부 이미지 기준 2~4번줄에서 냅니다. 셋잇단음표 괄호는 각 코드의 첫 세 이벤트, 즉 X → 음 → X 아래에 표시했습니다. 실제 악보의 줄 위치가 다르면 HTML 안의 openMidi 값만 바꾸면 됩니다.
