트라이어드·7th 쉘 보이싱 실시간 분석

Triad + Shell Voicing Ver.

트라이어드와 7th 쉘 보이싱을 함께 보는 지판 학습 도구

기존 트라이어드(R-3-5 / R-b3-5)에 더해, maj7·7·m7·mMaj7 같은 7th 계열을 3음 쉘 보이싱(R/3/7 중심)으로 표현합니다. 5도를 생략한 실전형 보이싱과 실제 연주 가능한 프렛 위치를 자동 계산합니다.

보이싱 설정기 (Voicing Builder)

Shell Voicing은 보통 5도를 생략하고 Root + 3도 + 7도만으로 maj7, 7, m7의 성격을 표현합니다.

트라이어드 / 쉘 보이싱 코드 폼

G(3)현과 B(2)현의 튜닝 변조를 100% 반영한 비주얼 격자 가이드라인입니다. 대표 Root 프렛을 입력하면 해당 프렛에 가까운 폼을 우선 표시하고, 원 안에는 실제 음이름(C, E, B 등)을 표시합니다. 미입력 시에는 도수(R, 3, b7 등)를 표시합니다. 하단에는 현재 선택한 코드타입의 코드폼만 표시됩니다. 개별 카드를 클릭하면 해당 보이싱으로 메인 지판이 전환되고 코드 사운드가 재생됩니다.
